В репозитории добавлены правила для маршрутизации v2ray.

# Установка v2ray:
## Linux, win, macOs
идем по ссылке https://github.com/2dust/v2rayN/releases
и выкачиваем последний релиз.
## Android
Для андроида придётся скачивать *.apk:
https://github.com/2dust/v2rayNG

# Добавление правил
Идем в настройки маршрутов - добавить.
В новом маршруте - добавить из файла.
Выбираем файл `all_rules.json`
